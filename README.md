# 痛打落水狗 (Tòng dǎ luò shuǐ gǒu)

> *"beat the dog that has fallen into the water"* — show no mercy to systems that are already collapsing.

**Tòng dǎ luò shuǐ gǒu** is transitioning from a minimal static HTML page into a full-fledged **De-dollarization & Resilience Web App Hub** (formerly conceptualized as *ResiHub*). It serves as the self-mirroring resilience and open-rails hub for creators, NGOs, and SMEs operating in BRICS+ and emerging markets.

## Vision & Architecture (2026 Update)

The project has been migrated to an **Astro + Cloudflare** static-first architecture. It is designed to be a living system that publishes itself on permanent storage and guarantees 100+ years of permanence while supporting machine-to-machine enterprise-grade payments.

### Key Pillars

1. **Self-Mirroring Resilience (Hybrid Stack)**
   - **IPFS + Filecoin Onchain Cloud:** The hub automatically mirrors itself upon deployment.
   - **Arweave + AO:** Pay once, store forever with an added compute layer.
   - Every page features a one-click "Mirror this page to IPFS + Filecoin + Arweave" capability.

2. **De-dollarized & Crypto Rails**
   - **x402 Foundation:** Native support for programmatic, machine-to-machine payments, unlocking premium APIs and datasets.
   - **BRICS Pay Integration:** Templates and guides for the decentralized payment messaging system, providing bridges for cross-border local currency flows.
   - **Lightning Zaps:** Support for direct tipping in sats via Nostr.

3. **Country Intelligence & Offline-First UX**
   - **ResiScore:** A dynamic widget evaluating storage permanence, payment rails, censorship risk, and connectivity for emerging markets.
   - **PWA Offline Cache:** Extreme low-bandwidth mode ensuring the entire hub loads in under 80 KB.
   - **Multi-language Support:** 11 built-in languages with edge-based localization.

## Tech Stack

- **Framework:** Astro 5
- **Deployment:** Cloudflare Pages & Workers (with built-in x402 support)
- **Content:** MDX / Markdown
- **Search:** Pagefind (client-side, zero-server)
- **Decentralization:** IPFS, Filecoin, Arweave, Nostr

## Getting Started

Install dependencies and run the local development server:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
